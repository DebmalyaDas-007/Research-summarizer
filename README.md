# 📄 Research Summarizer

A simple **AI-powered research paper and text summarizer** built using **LangChain, Google Gemini, and Streamlit**.
This application allows users to input research text or topics and generate concise summaries instantly.

---

## 🚀 Features

* ✨ Summarize research papers quickly
* 🤖 Powered by **Google Gemini LLM**
* 🔗 Built with **LangChain**
* 🖥️ Interactive UI using **Streamlit**
* ⚡ Fast and lightweight summarization
* 🧠 Adjustable creativity using temperature settings

---

## 🛠️ Tech Stack

* **Python**
* **LangChain**
* **Google Gemini API**
* **Streamlit**
* **dotenv**

---

## 📦 Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/DebmalyaDas-007/Research-summarizer.git
cd Research-summarizer
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
```

Activate it:

**Linux / Mac**

```bash
source venv/bin/activate
```

**Windows**

```bash
venv\Scripts\activate
```

---

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Add environment variables

Create a `.env` file in the root directory:

```
GOOGLE_API_KEY=your_google_api_key
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The app will open in your browser.
